
# Universal News Dashboard

## Project Overview

The **Universal News Dashboard** is a data-driven web application built using **Streamlit** that fetches the latest general news articles from various sources. The application provides users with up-to-date news in a simple, interactive format. The core functionality includes displaying news headlines, articles, and summaries, all fetched from a reliable news API. 

The primary goal of this project is to provide users with easy access to current global news, enabling them to stay informed about the latest happenings around the world.

## Features

- **Fetch Latest News**: Displays the most recent general news articles.
- **Simple Interface**: User-friendly and minimalist design.
- **Categorized News**: Displays news in an organized way, based on the source.
- **Real-Time Updates**: Fetches real-time news through an API integration.
  
## Technologies Used

- **Streamlit**: Used for building the web application interface.
- **Python**: The primary language for coding the backend functionality.
- **News API**: Fetches the latest news articles from different sources.
- **GitHub**: For version control and code repository management.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/amiEtegan/climate-change-dashboard.git
cd climate-change-dashboard
```

2. Install required libraries:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```
